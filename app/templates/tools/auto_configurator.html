Companion app required

This tool needs a small Mac companion app (Mac Admin Toolbox) that runs on your Mac and reads Privacy (TCC) data. The companion app must have Full Disk Access so it can read system permission settings.

  1. Download the companion app (zip) using the button below.
  2. Unzip the file (double‑click the .zip). You’ll get Mac Admin Toolbox.app.
  3. Move Mac Admin Toolbox.app to Applications (or keep it anywhere on your Mac).
  4. Open the app once so macOS registers it. It will open the Mac Admin Toolbox in your browser. The app is not notarized, so macOS may block it the first time. If so, go to System Settings → Privacy & Security, scroll down to the message about the app being blocked, and click Open Anyway. Then open the app again.
  5. In System Settings → Privacy & Security → Full Disk Access, click + and add Mac Admin Toolbox.
